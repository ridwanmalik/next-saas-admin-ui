# Getting Started

A production-ready Next.js SaaS admin dashboard template built with Tailwind CSS v4 and shadcn/ui v4.

## Prerequisites

- Node.js 20+
- Yarn 4 (configured via Corepack)

## Installation

```bash
yarn install
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) â€” you'll be redirected to `/dashboard` automatically.

## Tech Stack

| Layer | Tool |
|---|---|
| Framework | Next.js 16 (App Router, Turbopack) |
| Styling | Tailwind CSS v4 |
| UI Components | shadcn/ui v4 (new-york style) |
| Icons | Lucide React |
| Dark mode | next-themes |
| Language | TypeScript (strict) |

## Scripts

```bash
yarn dev       # Start dev server
yarn build     # Production build
yarn start     # Serve production build
yarn lint      # Lint with ESLint
```
