## 1. BerryDashboard (berrydashboard.com)

Berry is a **React + Material UI** admin dashboard by CodedThemes.

### Free Version (9 pages)
- Dashboard (Default)
- Sample Page
- Basic UI components (Buttons, Shadows, Typography, Color)
- Authentication (Login, Register, Forgot Password)

### Premium Version (45+ pages)
**Core differentiators over free:**
- Dark/Light mode toggle
- TypeScript version included
- Next.js seed version
- 6+ color theme options
- RTL support
- Multi-language (i18n)
- Design files (Figma)
- Multiple auth integrations: JWT, Firebase, Auth0, AWS Cognito, Supabase

**Apps & Feature Pages (premium):**
- Multiple Dashboard variants
- Analytics Dashboard
- E-Commerce Dashboard
- Social / Chat App
- Calendar App
- Kanban Board
- Mail Inbox
- User Management (list, profile, account)
- Invoice pages
- Pricing pages
- FAQ / Help pages
- Forms (validation, wizards, plugins — date picker, mask, autocomplete)
- Advanced Components (tree view, timeline, autocomplete, charts — ApexCharts)
- Authentication flows (JWT, OAuth, social login)
- Error pages (404, 500, under construction)
- Widgets page
- Landing page

**Technology Stack:**
- React 19, Material UI v7
- React Router, Redux + Context API
- Vite, Remix variants
- React Hooks API

**Other framework versions available:** Angular, Bootstrap, Vue (separate products)

## 3. Must-Have Features for Premium Admin Templates (Industry Consensus)

### Dashboard Variants (minimum 2–3)
- Default/Overview dashboard (KPI cards, charts, activity feed)
- Analytics dashboard (traffic, conversion, funnel charts)
- E-Commerce dashboard (revenue, orders, products)
- CRM/SaaS dashboard

### Authentication Pages
- Sign In / Login
- Sign Up / Register
- Forgot Password / Reset Password
- Two-Factor Authentication (2FA)
- Email verification flow

### User Management
- Users list (with search, filter, pagination, bulk actions)
- User profile page (avatar, bio, stats)
- User edit / account settings
- Team members / team management
- Roles & permissions

### Data Pages
- Data tables (sortable, filterable, paginated, exportable)
- Charts page (line, bar, pie, area, scatter, sparklines)
- Analytics/Reports with date range filters

### Forms
- Basic forms (inputs, selects, checkboxes, radios)
- Advanced forms (date picker, masked input, autocomplete, tags input, rich text)
- Form validation (with error states)
- Multi-step wizard/stepper
- File upload (drag and drop)

### App Pages
- Calendar (month/week/day views)
- Chat / Messaging UI
- Kanban / Task board
- Mail / Inbox
- File manager
- Invoice (view + print)
- Todo list
- Notifications center

### Account & Settings
- Profile settings
- Account preferences
- Notification settings
- Security settings (password change, 2FA, sessions)
- Billing / subscription page
- API keys / integrations

### Content & Marketing Pages
- Pricing page (cards + table variants)
- FAQ / Help center
- Landing page / marketing page
- Blog / article list and detail
- Gallery / photo grid
- Search results

### UI Component Showcase
- Buttons (all variants, sizes, states)
- Cards (basic, advanced, stat cards)
- Badges, alerts, toasts
- Modals, drawers/offcanvas
- Tabs, accordions, pagination
- Avatars, breadcrumbs
- Progress bars, spinners, skeletons/placeholders
- Typography, colors swatch page
- Icons page

### Error & Utility Pages
- 404 Not Found
- 500 / Server Error
- Under Construction / Maintenance
- Empty state
- Cookie consent banner
- Page loader

### Layout Options
- Vertical sidebar (standard)
- Horizontal top navigation
- Compact/mini sidebar
- Dark sidebar
- Dark mode (full)
- RTL support
- Condensed/boxed layout

### Plugins / Integrations
- Chart library (ApexCharts, Chart.js, or Recharts)
- Data table library (TanStack Table or equivalent)
- Date picker
- WYSIWYG / rich text editor
- Map (Leaflet or Google Maps)
- Drag and drop
- Full calendar
